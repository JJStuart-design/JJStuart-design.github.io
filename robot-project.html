Self-Balancing Robot Project

Tracking the progress of my embedded systems project - A two-wheeled self-balancing robot using STM32 and MPU-6050

Current Progress

Design Phase Completed

I've designed a tall, two-tiered robot chassis. The key is a low center of gravity, achieved by mounting the heavy battery pack below the main frame. It's powered by two high-torque geared motors and uses a PID control loop with data from a gyroscope and motor encoders to balance in place.
